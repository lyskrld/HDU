mapcache-utils
--------------

* To run the mapcache utils, you must have the /ms4w/Apache/cgi-bin directory
  included in your PATH.

  * run /ms4w/setenv.bat to include this directory in your path,
    or set it in your Environment Variables on your machine

* see https://mapserver.org/mapcache/seed.html
      https://mapserver.org/mapcache/coverage.html
  for usage documentation.
