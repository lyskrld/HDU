* To run openssl.exe, you must have the /ms4w/Apache/cgi-bin directory
  included in your PATH.

  * run /ms4w/setenv.bat to include this directory in your path,
    or set it in your Environment Variables on your machine
    
    * setenv.bat also sets the required variable OPENSSL_CONF

Documentation: https://www.openssl.org/docs/man1.1.1/
