* To run the NetCDF utilities, you must have the /ms4w/Apache/cgi-bin directory
  included in your PATH.

  * run /ms4w/setenv.bat to include this directory in your path,
    or set it in your Environment Variables on your machine

Documentation: https://www.unidata.ucar.edu/software/netcdf/docs/netcdf_utilities_guide.html
